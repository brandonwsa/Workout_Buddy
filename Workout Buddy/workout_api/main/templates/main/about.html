Welcome to Workout Buddy!

What you can do:


This web application allows a user to register up an account and to log their workouts.

When a user creates their account, they can create exercises and the exercise details associated with that exercise to their workout.

The user can then also update, delete, or add more exercises/exercise details to their workout or create a new workout.

What is handy about this web app is that the total volume for your exercise will be calculated automatically so you don't have to do it yourself. Don't worry, the volume will be adjusted if you make any updates to your exercise details as well too.

How it works:


Say you did a Full Body A workout routine Monday, log that down and the database will save all the information so you can view it anywhere. Once you do Full Body A again on Friday, just create a workout with the same name if you'd like, and you can go back to the Full Body A you did on monday to compare the two.
